# Project Media Files

## Folder Structure

### Videos (`/videos`)
Place your project demo videos here. Supported formats: `.mp4`, `.webm`, `.mov`

**Naming Convention:**
- `muse-sketch-studio.mp4`
- `uw-chatbot.mp4`
- `custom-chatbot.mp4`
- `low-stock-alert.mp4`
- `portfolio.mp4`
- `caraksha.mp4`
- `gim.mp4`
- `imoney.mp4`

### Images (`/images`)
Place your project screenshots/thumbnails here. Supported formats: `.png`, `.jpg`, `.webp`

**Naming Convention:**
- `muse-sketch-studio.png`
- `uw-chatbot.png`
- `custom-chatbot.png`
- `low-stock-alert.png`
- `portfolio.png`
- `caraksha.png`
- `gim.png`
- `imoney.png`

## How to Add Media

1. Copy your video/image files to the appropriate folder
2. Use the naming convention above
3. The website will automatically display them in project cards!

## File Size Recommendations

- **Videos**: Keep under 10MB for fast loading (use compression if needed)
- **Images**: 1920x1080 or smaller, optimized for web
